Static Site Generators (SSG) convert documents, often markdown, to [HTML](html.md) and deploy them to a web server. They differ from dynamically generated sites that use PHP such as Wordpress.

- [Jekyll](http://jekyllrb.com/)
- [Hugo](hugo.md)
- [Hakyll](https://github.com/jaspervdj/Hakyll/)
- [yst](https://github.com/jgm/yst)
- [Nanoc](https://github.com/nanoc/nanoc)
