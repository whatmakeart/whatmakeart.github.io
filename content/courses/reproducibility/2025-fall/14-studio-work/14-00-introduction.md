## Overview

Studio work and reclaiming screen print screens. This is an important process in an individual studio and especially important in a shared studio with shared print screens. Remaining class time is spent completing and photo documenting student produced zines.

## Learning Objectives

- Properly remove emulsion from print screen and reclaim for reuse
- Follow through with print screen loan agreement by returning an intact print screen
