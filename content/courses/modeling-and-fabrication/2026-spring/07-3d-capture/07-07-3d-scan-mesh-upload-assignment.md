## Assignment Deliverables

1. Upload exported 3D mesh of 3D scan of object
   - Label file YYYY-MM-DD Lastname Firstname 3D Scan Mesh (`.obj`)
2. Upload exported diffuse image texture file from 3D scan of object
   - Label file YYYY-MM-DD Lastname Firstname 3D Scan Texture (`.png`, `.jpg`)
3. Upload exported material file from 3D scan of object
   - Label file YYYY-MM-DD Lastname Firstname 3D Scan Material (`.mtl`)

## Assignment Overview

[3D scanning](../../../../3d-modeling/3d-scanning.md) is a quick way to capture 3D geometry from the physical world for use in digital 3D modeling programs.

### Process

1. Use a structured light scanner at the IML or the think\[box\] to capture the form of a 3D physical object
   - the IML has Einscan scanners
   - the think\[box\] has an Artec Leo scanner
2. Process the scan data with the either Einscan's or Artec's capture software.
3. Export an OBJ mesh file, image texture file, and a material file.
4. Label the files.
5. Upload the files to the assignment page. _If the mesh file is too large to upload, then you can optimize it to make a smaller file size or upload to a cloud file sharing service and share the link._

#### Optional Process

3D scans need to be cleaned up and optimized for use. You can explore cleaning up your 3D scans using Blender or other software following the guides below. This is not required for the assignment but is encouraged.
