Portfolio Website Weekly Review / Intro

Each week there will be video(s) that go over everything you need to know to set up your portfolio websites by the end of the semester. If you do the exercises each week, by the end of the semester you will have a portfolio website on a custom domain name. You can then add documentation of the projects that you make this semester to your portfolio. These videos are designed to start from zero knowledge but can be used as reviews / refreshers also.

If you did not make a website with a custom domain last semester, it is especially important that you do the two videos below. This creates a GitHub account and gets you a free domain name. Sometimes it takes a bit to process the domain name so best to do it earlier than later
